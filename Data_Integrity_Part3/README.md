# CodeJudge Platform - Data Integrity Audit & Staging Repair Workflow

This repository holds the compliance, data validation, domain rule checking, and staging table repair configuration scripts for the CodeJudge platform data migration phase.

## 📁 Repository Contents
* `import_validation.sql` - Row verification queries.
* `integrity_audit.sql` - Primary, unique, and foreign key isolation validations.
* `domain_rule_checks.sql` - Business rules evaluation suite.
* `repair_plan.md` - Strategy architecture map.
* `staging_repair_scripts.sql` - Non-destructive isolated data fix execution logs.
* `before_after_evidence.md` - System stability verification table.
